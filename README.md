# Professional Video Editor (Python)

A comprehensive video editing application built with Python and PyQt6.

## Features

- Timeline-based video editing
- Multi-track support
- Video effects (brightness, contrast, saturation, etc.)
- Audio controls (volume, speed, fade)
- 6 beautiful themes
- Export to multiple formats (MP4, AVI, MOV)
- Project save/load functionality

## Requirements

- Python 3.8 or higher
- See requirements.txt for dependencies

## Installation

```bash
pip install -r requirements.txt
```

## Building Executable

```bash
python build_executable.py
```
